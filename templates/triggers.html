Triggers

En trigger betyder inte att du är svag eller överreagerar.
Ibland reagerar nervsystemet på något som påminner om tidigare upplevelser, gamla känslor eller roller vi lärt oss att ta.
Kroppen försöker ofta skydda oss långt innan det logiska tänkandet hinner förstå vad som händer.

Vanliga tecken på att något triggat dig

  • Stark känsloreaktion
  • Svårt att tänka klart
  • Behov av kontroll
  • Vilja att fly eller stänga av
  • Överanalysering
  • Stark ilska eller skam
  • Rädsla för att bli avvisad
  • Känslan av att vara "liten igen"

Tar ofta ansvar för andras känslor och försöker lösa situationer för att skapa lugn eller undvika konflikt.

Drar sig undan, håller tillbaka sina känslor eller säger inget för att skydda sig själv eller undvika att bli sårad.

Försöker göra rätt, prestera eller vara till lags för att känna sig trygg, uppskattad eller tillräcklig.

Försöker förstå varje detalj och tänka igenom allt för att få kontroll och minska osäkerhet eller obehag.

Undviker situationer, känslor eller konflikter genom att dra sig undan, stänga av eller vilja komma bort från det som känns jobbigt.

Tar ofta på sig skuld, ansvar eller press även för sådant som egentligen inte ligger på ens egna axlar.

Kan känna stark rädsla för avstånd, avvisande eller att bli lämnad och reagerar ofta utifrån den rädslan.

Reagerar med irritation, försvar eller ilska när något känns hotfullt, orättvist eller känslomässigt smärtsamt.
